Chosen组件

Chosen 是一个支持jquery的select下拉框美化插件，它能让丑陋的、很长的select选择框变的更好看、更方便。不仅如此，它更扩展了select，增加了自动筛选的功能。它可对列表进行分组，同时也可禁用某些选择项。

        js及css引入：
  <link href="../js/plugins/chosen/css/chosen-min.css" rel="stylesheet">
  <script src="../js/plugins/chosen/jquery.choosen.concat-min.js"></script>
  <script src="../js/utils/jquery.utils.pinyin-min.js"></script>
  <script src="../js/jquery.utils.contact-min.js"></script>

select：
  <s:select name="shzt_cx" id="shzt_cx" headerValue="--全部--" cssClass="form-control chosen-select" style="display: none;"></s:select>

select对应的html代码：
  <select name="shzt_cx" id="shzt_cx" class="form-control chosen-select" style="display: none;">
    <option value="">全部</option>
    <option value="0">待确认</option>
    <option value="1">待审核</option>
    <option value="2">审核中</option>
    <option value="3">已通过</option>
    <option value="4">被退回</option>
    <option value="5">终止</option>
  </select>

初始化：
  $(".chosen-select").chosen();

jqGrid表格

jqGrid 是一个在jquery基础上做的一个表格控件，以ajax的方式和服务器端通信。 查看demo详情

css与js引入：

  <script type="text/javascript" src="../js/plugins/jqGrid4.6/jquery.jqgrid.src-min.js" charset="utf-8"></script>
  <script type="text/javascript" src="../js/plugins/jqGrid4.6/jquery.jqgrid.contact-min.js" charset="utf-8"></script>
  <script type="text/javascript" src="../js/plugins/jqGrid4.6/jquery.jqgrid.settings.js" charset="utf-8"></script>
  <script type="text/javascript" src="../js/plugins/jqGrid4.6/jquery.jqGrid-min.js" charset="utf-8"></script>

bootbox弹窗插件

bootbox 是一个JavaScript库，它允许你使用自举模式创建方案对话框，而不必担心创建、管理或删除任何所需的DOM元素或者JS的事件处理程序。 查看demo详情

注意：此版本已经对原始bootbox进行了封装。

css与js引入：
  <link rel="stylesheet" type="text/css" href="../js/plugins/bootbox/css/bootbox-min.css">
  <script type="text/javascript" src="../js/plugins/bootbox/bootbox.concat-min.js?v=20220905v1" charset="utf-8"></script>
js代码：
  $("#example_alert").click(function () {
      $.alert("alert方式弹出框");
      return false;
    });

    $("#example_success").click(function () {
      $.success("success方式弹出框");
      return false;
    });

    $("#example_error").click(function () {
      $.error("error方式弹出框");
      return false;
    });

    $("#example_prompt").click(function () {
      bootbox.prompt({
        title:"您的大名？",
        value:"张老板",
        callback: function (result) {
          $.alert("返回："+result);
        }
      });
      return false;
    });

    $("#example_confirm").click(function () {
      $.confirm("confirm方式弹出框",function(isBoolean){
        $.alert("返回："+isBoolean);
      });
      return false;
    });


    $("#example_exprot").click(function () {
      $.showDialog('example-export.html', '导入', {
        width : "900px",
        modalName : "addModal",
        buttons : {
          success : {
            label : "下一步",
            className : "btn-primary btn-next-step",
            callback: function () {
              $.showDialog("example-export-setup.html","导入设置",{
                width : "900px",
                modalName : "addModal",
                buttons : {
                  success: {
                    label: "确定导入",
                    className: "btn-primary"
                  },
                  cancel : {
                    label : "关　闭",
                    className : "btn-default"
                  }
                }
              });
              return false;
            }
          },
          cancel : {
            label : "关　闭",
            className : "btn-default"
          }
        }
      });
    });
  });

Validation验证插件

这个jQuery插件使简单的客户端表单验证更加方便，同时还提供大量的定制选项。 查看详情

必填、验证有效性、最长200。data-rules='{"required":true,"stringMaxLength":200}'
必填、最小长度5。data-rules='{"required":true,"stringMinLength":5}'
必填。data-rules='{"required":true}' 
css与js引入：
    <link rel="stylesheet" type="text/css" href="../js/plugins/validation/css/jquery.validate-min.css">
    <script src="../js/plugins/validation/jquery.validate-min.js"></script>
    <script src="../js/plugins/validation/jquery.validate.contact-min.js"></script>
    <script src="../js/plugins/validation/jquery.validate.methods.contact-min.js"></script>
    <script src="../js/plugins/validation/lang/zh_CN-min.js"></script>

html代码：
    <form class="form-horizontal" role="form" data-bs-toggle="validation" id="ajaxForm" name="ajaxForm" action="">
        <input type="email" class="form-control" id="inputEmail3" name="inputEmail3" placeholder="邮箱" data-rules='{"required":true,"stringMaxLength":200}'>
        <input type="text" class="form-control" id="userName" name="userName" placeholder="用户名" data-rules='{"required":true,"stringMinLength":5}'>
    </form>

js代码：
    $('[data-bs-toggle*="validation"]').trigger("validation");
